# Contributing

Contributions to the Insider Threat Analyst Career Framework are welcome.

## How to contribute

- **Suggest page** — https://insiderthreatanalyst.com/suggest.html
- **Email** — hello@insiderthreatanalyst.com
- **Pull request** — against this repository

## Credit is opt-in

Contributors are named on the site only with their consent. This is existing
practice: if you'd like public credit, say so, and it will be added to
`/contributors.html`. If you'd rather remain unnamed, your contribution is
still welcome and appreciated.

## Working with the code

- The site is plain static **HTML/CSS/JS** — there is no build step and no
  framework to install. Open the files directly.
- Keep pull requests small and scoped to a single change so they're easy to
  review.
